# AI Mastery Course

A comprehensive 4-day in-person AI course for technical professionals. From AI basics to building custom AI assistants.

## Course Overview

| Day | Topic | Focus |
|-----|-------|-------|
| 1 | AI Foundations & Productivity | ChatGPT, Claude, Gemini, prompt engineering |
| 2 | AI-Powered Development | Cursor, Copilot, Claude Code, vibe coding |
| 3 | Documentation & LLM Mastery | AI docs, model comparison, cost optimization |
| 4 | Building AI Assistants | Custom GPTs, Claude Projects, Clawdbot |
