üí° Prompting Tips

Be Specific

Vague prompts get vague results. Include context, constraints, and desired format.

Use Examples

Show don't tell. One example is worth 100 words of description.

Define Output Format

Want JSON? Say so. Want bullets? Specify. AI follows explicit format requests.

Iterate

First response not perfect? Refine. "Make it shorter" or "Focus more on X".
