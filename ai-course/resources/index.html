Day 1 Setup

Accounts Needed

  • ChatGPT account (free or Plus)
  • Claude account (free or Pro)
  • Gemini account (free with Google)

Recommended

  • ChatGPT Plus ($20/mo) for GPT-4 access
  • Claude Pro ($20/mo) for higher limits

Day 2 Setup

Required Tools

  • Cursor (free trial available)
  • VS Code (if not using Cursor)
  • GitHub Copilot ($10/mo)

Optional

  • Claude Code (Anthropic API key)
  • Node.js 18+ for exercises

Day 4 Setup

  • Claude Pro (for Projects)
  • Text editor for SOUL.md files

Advanced (Optional)

  • Anthropic API key
  • OpenAI API key
  • Clawdbot installation
